Implementation Notes

CSS Classes

  • header-variant-1: Standard layout with logo on left
  • header-variant-2: Centered logo layout
  • header-variant-3: Grid layout for admin dashboards
  • header-variant-4: Practitioner style with white background
  • header-variant-5: Compact header without navigation
  • header-variant-6: Icon-based navigation
  • mobile-header: Mobile responsive header

Color Scheme

  • Primary: #0b7285 (Teal)
  • Primary Dark: #095a69
  • Background Soft: #e5f3f7
  • Header Text: #ffffff

Responsive Breakpoints

  • Mobile: max-width: 768px
  • Tablet: 769px - 1023px
  • Desktop: min-width: 1024px

JavaScript Requirements

For mobile menu functionality, you'll need JavaScript to toggle the mobile-nav.active class and handle menu open/close states.
