# Assets Folder

This folder is for storing your application assets like logos, images, and icons.

## Logo Files

Place your logo files here with the following naming convention:

- `logo.png` - Main logo (recommended size: 200x60px)
- `logo-white.png` - White version for dark backgrounds (recommended size: 200x60px)
- `logo-small.png` - Small version for headers (recommended size: 100x30px)
- `favicon.ico` - Browser favicon (recommended size: 32x32px)
- `apple-touch-icon.png` - iOS home screen icon (recommended size: 180x180px)

## Supported Formats

- PNG (recommended for logos)
- JPG (for photos)
- SVG (for scalable graphics)
- ICO (for favicons)

## Usage in Application

The logos will be automatically integrated into the application when you place them in this folder.
